» Installation Instructions
  1. Open Root Wordpress Folder
  2. Put:
    File: wap.php
    File: wap-comments.php
  3. Refer To Usage For Further Instructions
  4. You DO NOT need to activate WP-Wap
» Upgrade Instructions
From v1.0x To v2.30
  1. Open Root Wordpress Folder
  2. Put/Overwrite:
    File: wap.php
    File: wap-comments.php
  3. Delete:
    File: wp-wap.php
    File: wp-wap-comments.php
  4. Refer To Usage For Further Instructions
  5. You DO NOT need to activate WP-Wap
